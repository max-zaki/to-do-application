# ToDo App
A simple interactive command line program to create, edit, and view a To Do list.

## How to view this project:
### Clone or download the following repository:
https://github.com/max-zaki/to-do-application.git

## How the program works:
On running the program, the user will be greeted and prompted for their name.  They will then be shown a menu from which to choose a command to give.  The following are the menu options:

* View tasks
* Add a task
* Delete a task
* Quit

Invalid input is gracefully handled by the program.
